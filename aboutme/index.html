About Me

Technical Profile

Cybersecurity Research & Labs

I actively engage in hands-on security labs focused on Linux systems administration, Nginx server hardening, and network traffic analysis. I specialize in documenting the technical "why"—translating complex packet captures and firewall configurations into actionable security insights.

Modern Development Workflow

My projects are built within VS Code using a hybrid development model. I combine raw logic with AI-integrated workflows, utilizing Claude and GitHub Copilot to accelerate boilerplate tasks while maintaining a strict manual audit for security and code integrity.
