Labs & Projects

My labs focus on practical security implementations, including server hardening with Nginx, Linux system configuration, and network traffic analysis using tools like Wireshark. Each project emphasizes not only execution, but also analysis—breaking down system behavior, identifying vulnerabilities, and documenting security decisions.

Alongside security labs, I develop software projects to strengthen core computer science fundamentals. My development workflow is centered in VS Code and follows a hybrid approach: combining first-principles coding with AI-assisted tools such as Claude and GitHub Copilot to improve efficiency, while maintaining full control and manual validation of all logic and security considerations.
